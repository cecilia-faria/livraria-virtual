# Livraria Virtual

Projeto de loja virtual de livros desenvolvido com HTML, CSS e JavaScript puro. A aplicação simula um catálogo de livros com carrinho de compras funcional e páginas individuais para cada produto.

# Demonstração

A página principal exibe um carrossel de banners e um grid responsivo de produtos. Ao clicar em um livro, o usuário é redirecionado para a página de detalhes do produto.

# Funcionalidades

- Grid responsivo de produtos (CSS Grid + Flexbox)
- Carrossel de banners com navegação por botões
- Carrinho de compras lateral com animação
- Controle de quantidade de itens no carrinho
- Cálculo automático do total em reais
- Páginas individuais de produto com galeria de imagens
- Layout responsivo para mobile

# Tecnologias Utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — Grid Layout, Flexbox, transições e animações
- **JavaScript** — manipulação do DOM, eventos e lógica do carrinho

# Como Executar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` no seu navegador
3. Nenhuma instalação ou dependência necessária
